REM Title: Install Wireshark via Chocolatey (Admin CMD)
REM Author: SoggyCow
REM Description: Assumes Chocolatey is pre-installed; installs Wireshark silently and closes terminal

DELAY 1000
GUI r
DELAY 500
STRING cmd
CTRL-SHIFT ENTER
DELAY 1500
STRING choco install wireshark -y && exit
ENTER
