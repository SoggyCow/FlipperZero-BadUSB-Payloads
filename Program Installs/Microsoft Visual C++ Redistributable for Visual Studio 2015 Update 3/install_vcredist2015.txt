REM Title: Install Microsoft Visual C++ Redistributable for Visual Studio 2015 Update 3 via Chocolatey (Admin CMD)

REM Description: Assumes Chocolatey is pre-installed; installs Microsoft Visual C++ Redistributable for Visual Studio 2015 Update 3 silently and closes terminal
