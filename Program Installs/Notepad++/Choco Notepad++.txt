REM Title: Install Notepad++ via Chocolatey (No Choco Install)
REM Author: SoggyCow
REM Description: Assumes Chocolatey is pre-installed; installs Notepad++ silently

DELAY 1000
GUI r
DELAY 500
STRING powershell -NoP -W Hidden -Command "Start-Sleep -Seconds 1; choco install notepadplusplus.install -y"
ENTER
