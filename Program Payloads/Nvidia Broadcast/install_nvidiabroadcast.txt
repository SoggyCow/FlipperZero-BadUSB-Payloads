REM Title: Install Nvidia Broadcast via Chocolatey (Admin CMD)
REM Author: SoggyCow
REM Description: Assumes Chocolatey is pre-installed; installs Nvidia Broadcast silently and closes terminal

DELAY 1000
GUI r
DELAY 500
STRING cmd
CTRL-SHIFT ENTER
DELAY 1500
STRING choco install nvidia-broadcast -y && exit
ENTER
